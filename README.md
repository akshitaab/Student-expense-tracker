## 📌 Overview

| | |
|---|---|
| **Goal** | Help students track spending habits and manage budgets |
| **Interface** | Interactive Streamlit web app |
| **Input** | Daily expense entries with category and amount |
| **Output** | Visual summaries, charts, and exportable records |

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| ➕ Add Expenses | Log daily expenses with category, amount, and date |
| 📊 Category Pie Chart | Visual breakdown of spending by category |
| 📈 Daily Bar Graph | Day-wise expense trend |
| 🧾 Expense Table | Full record of all entries |
| 💰 Total Summary | Running total of spending |
| 📥 CSV Export | Download your expense data |

## 🚀 Run Locally

**1. Clone the repository**
```bash
git clone https://github.com/akshitaab/student-expense-tracker.git
cd student-expense-tracker
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Launch the app**
```bash
streamlit run app.py
```
