# 🎓 Student Expense Tracker

A simple and beginner-friendly web app to track your daily expenses, visualize your spending habits, and manage your budget – built with Python and Streamlit.

---

## 💡 Features
- Add daily expenses with category and amount
- View total spending
- Category-wise pie chart
- Daily expense bar graph
- All expense records in a table

---

## 🛠️ Tech Stack
- Python 🐍
- Streamlit 📊
- Pandas 📑
- Matplotlib 🎨
